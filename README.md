# Simulasi Game of Life (Array 2D)

Program ini merupakan simulasi Game of Life yang dibuat oleh John Conway.

## 📖 Aturan:
1. Sel hidup dengan kurang dari 2 tetangga → mati
2. Sel hidup dengan 2 atau 3 tetangga → tetap hidup
3. Sel hidup dengan lebih dari 3 tetangga → mati
4. Sel mati dengan tepat 3 tetangga → menjadi hidup

## 🛠 Struktur Data
Program menggunakan:
- Array 2 Dimensi (List of List)
